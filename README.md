# React-Webpack-Template

## Introduction

React project built on Webpack

## Features

- webpack5
- typescript(__Branch:typescript__)
- less
- eslint
- stylelint
- commitlint

## Getting Started

```javascript
# enter the project directory
cd react-webpack-template

# install dependency
yarn or npm install

# develop
yarn dev or npm run dev

```
